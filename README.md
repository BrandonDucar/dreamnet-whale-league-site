# Whale Intelligence League Site

This is the public product and launch site for [Whale Intelligence League](https://dreamnet-whale-league.pages.dev).

The site uses real screenshots from the live Svelte product. It does not claim live quotes, real trading, seeded human performance, affiliate relationships, or execution capability.

## Local development

```bash
npm install
npm run dev
```

## Verification

```bash
npm run lint
npm run build
npm test
```

## Product boundary

- This repository explains and launches the product.
- [`dreamnet-whale-league`](https://github.com/BrandonDucar/dreamnet-whale-league) is the interactive Svelte app.
- The first release is paper-only and includes one disclosed teaching agent.
